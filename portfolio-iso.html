GEPI

Información del proyecto

  • Categoría: Gestión
  • Fecha: Diciembre 2021

Trabajo de prácticas desarrollado para la asignatura Ingeniería del Software II.
Su objetivo es la gestión de pacientes con enfermedades infectocontagiosas. Se pide el desarrollo de un software para este fin, pero se hace hincapié en la parte de gestión del proyecto.
Con la colaboración de Diego Alba, María Victoria Alcázar, Jesús Calzado, Alejandro Panigua y Enrique Rubio.
